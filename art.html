Ping! Pong! Game!

This is the first project I built in TouchDesigner while teaching myself the software. The concept is a ping pong simulator where users can hit the ball back when it reaches their side of the table. The software detects the speed at which the user moves via the pc camera and uses this data to control the speed of the returning ball. I found it particularly interesting to implement the physics myself using Python within TouchDesigner.

A text gif produced with the same pipeline.
Testing out different gravity and speed parameters, as well as dynamic camera angles.
